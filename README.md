# PrivacyGuard
PrivacyGuard: Intelligent Privacy Policy AuditorPrivacyGuard AI is a sophisticated browser-integrated solution designed to bridge the transparency gap between complex legal jargon and consumer data rights. By leveraging a Flask-driven Python backend and a Manifest V3 Chrome Extension, the system performs real-time linguistic auditing of privacy policies to identify high-risk data practices.The platform utilizes a weighted keyword-matching algorithm to categorize risk into three distinct tiers: Low, Moderate, and High.
